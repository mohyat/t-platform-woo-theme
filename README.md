# T-Platform WooCommerce Theme

Profesionalna WooCommerce tema po vzoru Foxway.dk - B2B dizajn za tehnicne izdelke.

## Opis

T-Platform WooCommerce Theme je WordPress plugin ki "preoblece" katerokoli WooCommerce trgovino v profesionalen B2B dizajn podoben Foxway.dk.

### Kljucne znacilnosti:
- Foxway-inspired dizajn - cist, profesionalen B2B izgled
- Plugin arhitektura - namesti se na katerokoli WooCommerce trgovino
- Modularna struktura - loceni moduli za header, footer, product cards
- Custom produktne kartice - prikaz specifikacij (CPU, RAM, Storage)
- Quality badges - Teqcycle Pro, HP Certified, New Retail
- Responsive dizajn - deluje na vseh napravah
- Admin nastavitve - prilagodi barve, logotipe, layout

## Namestitev

1. Prenesi plugin ZIP datoteko
2. WordPress Admin -> Plugins -> Add New -> Upload Plugin
3. Aktiviraj plugin
4. Konfiguriraj nastavitve pod Settings -> T-Platform
